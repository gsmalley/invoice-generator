# Invoice Generator

A web-based invoice generation application.

## Repository Structure

### Branches
- **master** - Main production branch
- **develop** - Development branch for ongoing work
- **stage** - Staging environment for testing
- **prod** - Production-ready code

### Environment Ports

| Environment | Port | Description |
|-------------|------|-------------|
| Production | 5177 | Live production environment |
| Development | 5178 | Development environment |
| Stage | 5179 | Staging/QA environment |

### Feature Branches
Feature branches will be created as needed from the `develop` branch.
